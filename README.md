# MOE HATANAKA Portfolio
自身のポートフォリオサイトを Next.js で作成しました。
簡潔に見て頂けるよう、スッキリしたレイアウトを心がけました。

## 技術スタック
- **フレームワーク**: Next.js
- **CSS**: Tailwind CSS
- **アニメーション**: Framer Motion
- **画像**: Next.js Image

## 機能・特徴
- コンポーネント化して再利用性を確保
- `map` を使って作品を動的に表示
- レスポンシブデザイン対応（スマホ〜PC）
- アニメーションで見出しや作品が自然にフェードイン

## デザインコンセプト
- シンプルで見やすいUI/UXを意識
- 自分を表すサイトなので、自分の好きな空や太陽をモチーフにしました
